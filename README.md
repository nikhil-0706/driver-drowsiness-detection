# Driver Drowsiness Detection using OpenCV + SVM

A real-time driver drowsiness detection system built using OpenCV, Haar Cascade, and SVM.

## Features
- Real-time webcam eye detection
- Open / Closed eye classification using SVM
- Drowsiness alert alarm
- Screenshot capture during alert
- Alert logging

## Tech Stack
- Python
- OpenCV
- Scikit-learn
- NumPy
- Pickle

## Project Structure
- model_training.ipynb
- webcam_detection.ipynb
- models/
- screenshots/
